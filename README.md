Custom-IDE
----------
C# Framework for building custom IDE.
* Text Editor with keyword colors.
* Current Line Highlighting.
* Dynamic Highlighting with popup text.
* Loading and Saving of Theme and Editor Font
* Default Theme

--------

* Actual IDE for GLSL shaders.
